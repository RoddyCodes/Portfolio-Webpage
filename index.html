HERO Chatbot

A full-stack, AI-powered chatbot designed to connect individuals and frontline workers with affordable housing resources. This application features a modern microservice architecture, including a Next.js frontend, a primary backend API built with NestJS, and a dedicated Python LLM microservice using FastAPI. It leverages a Retrieval-Augmented Generation (RAG) pipeline to provide contextually-aware answers grounded in a knowledge base of housing information. To deliver the most current data, the chatbot also integrates with the live Bloom Housing API for real-time listings.
